# Build automation for GP2X-SDL version of phoneME

Pre-reqs:
- python3
- `pip3 install pexpect`
- imagemagick
- qemu-system-i386
- ssh
- sshpass
- debian-lenny.qcow2 (see blog post)

Run it:

	python3 main.py

Tool versions:
- Debian 5.0.10
- binutils 2.16.1
- gcc 3.4.6
- (host libc)
- SDL 1.2.15
- SDL_image 1.2.12
- SDL_mixer 1.2.12
- JDK 1.4.2_19
